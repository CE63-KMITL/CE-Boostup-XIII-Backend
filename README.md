## Setup

```bash
$ git clone https://github.com/CE63-KMITL/CE-Boostup-13-Backend
$ cd CE-Boostup-13-Backend
$ npm install
```

## Compile and run the project

### Development

```bash
$ npm run start
```

### Development (auto update, Watch mode)

```bash
$ npm run start:dev
```

### Production mode

```bash
$ npm run start:prod
```
